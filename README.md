# Devinette
Description
Jeu interactif où l'ordinateur génère un nombre aléatoire entre 1 et 100. L'utilisateur a 10 essais pour le trouver.

Règles
L'ordinateur choisit un nombre secret entre 1 et 100
L'utilisateur propose un nombre
Le programme indique si le nombre est trop grand ou trop petit
L'utilisateur gagne s'il trouve le nombre en 10 essais maximum
L'utilisateur perd s'il épuise ses 10 essais
Comment exécuter ?
Compiler : javac Main.java
Exécuter : java Main
